# Cognitive Engine (FSM & LLM Orchestrator)

## 1. Misión
Optimizar la asimilación de conocimiento y prevenir el colapso cognitivo. Es el guardián que regula cuánta información nueva puede procesar el usuario basándose en su estado biológico real.

## 2. Fundamentos Teóricos
- **Spaced Repetition (SRS) & Active Recall:** Basado en la curva del olvido de Ebbinghaus. El sistema programa repasos en intervalos óptimos para consolidar la memoria a largo plazo.
- **Máquina de Estados Finitos (FSM):** Orquesta el flujo de aprendizaje. Dependiendo del desempeño del usuario, el motor decide si avanzar a conceptos complejos, reforzar el estudio quirúrgico o retroceder a bases conceptuales (Regresión N-1).
- **Throttling Biométrico:** Utiliza datos de HRV y Sueño para calcular el `Umbral_Tsunami_Dinamico`. Si el estrés biológico es alto, el sistema detiene la captura de nuevos "Learning Nodes" para priorizar la consolidación.
- **Zettelkasten:** Método de notas atómicas y conectadas. El sistema incentiva la síntesis final de cada tema para su almacenamiento en el grafo de conocimiento.
