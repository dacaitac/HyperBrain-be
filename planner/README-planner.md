# Agenda Planner (Stochastic Bin-packing)

## 1. Misión
Transformar una lista priorizada en una agenda ejecutable en el calendario real. Es el puente entre la intención y el tiempo físico.

## 2. Fundamentos Teóricos
- **Knapsack Problem (Mochila):** Optimización combinatoria para llenar franjas horarias limitadas con la combinación de tareas que maximice el valor total.
- **Bin-packing Estocástico:** Acepta variabilidad en la duración real de las tareas, ajustando la programación según probabilidades de éxito.
- **Gestión de Energía:** Asigna tareas de alta carga cognitiva a las franjas de mayor energía (mañanas) y tareas mecánicas a franjas de decaimiento.
- **Margen de Caos:** Implementa la teoría de amortiguamiento (Buffer Management) reservando el 20% del tiempo para evitar el efecto dominó en caso de retrasos.
